# مهارة: تطوير ويب احترافي (مع RTL وطباعة)

## متى تُستخدم
عند بناء صفحة/تطبيق ويب بـ HTML/CSS/JS، أو ضبط الطباعة A4، أو دعم العربية RTL، أو تحسين الأداء والوصولية.

## HTML دلالي
- عناصر دلالية: `header/nav/main/section/article/aside/footer`، و`<h1>` واحد فقط وتسلسل عناوين منطقي.
- نماذج صحيحة: `<label>` مرتبط، `fieldset/legend`، أنواع إدخال مناسبة، رسائل خطأ مربوطة بـ `aria-describedby`.

## CSS حديث
- اعتمد **الخصائص المنطقية** بدل الفيزيائية: `margin-inline`, `padding-inline`, `inset-inline`, `text-align: start`.
- متغيرات CSS لرموز التصميم، Grid/Flexbox للتخطيط، وحدات مرنة (rem/%, clamp).
- حالات `:focus-visible` واضحة، واحترم `prefers-reduced-motion` و`prefers-color-scheme`.

## RTL عربي (وفق W3C)
- `<html dir="rtl" lang="ar">`، واستخدم `dir="auto"` لحقول الإدخال المتغيّرة المحتوى.
- اعتمد الترتيب المنطقي لا البصري، ودَع خوارزمية BiDi تعمل؛ تجنّب أكواد التحكم اليدوية.
- الجداول والتخطيط تنعكس تلقائياً مع الخصائص المنطقية؛ راعِ الأرقام والاتجاه المختلط.

## الطباعة A4 الدقيقة (مهم لهذا المشروع)
- `@page { size: A4; margin: <مطابق للأصل>; }` و`@media print` لإخفاء عناصر الواجهة وإظهار النموذج الرسمي فقط.
- امنع تقطيع الجداول/الأقسام: `break-inside: avoid;` و`break-after`/`break-before` عند الحاجة.
- ثبّت ألوان/حدود الطباعة بـ `print-color-adjust: exact;`، واضبط حجم الخط والهوامش لتطابق النموذج الورقي حرفياً.

## الأداء والوصولية
- Core Web Vitals: قلّل الحظر، حمّل الخطوط بكفاءة (`font-display: swap`)، صوّر/جزّئ الأصول.
- WCAG 2.2 AA: تباين، تنقل لوحة مفاتيح، أدوار/تسميات ARIA، نصوص بديلة.

## الإجراء (Procedure)
1. ابنِ هيكل HTML دلالي ونماذج صحيحة.
2. أنشئ نظام تصميم بمتغيرات CSS وخصائص منطقية (RTL-ready).
3. اضبط شيت طباعة `print.css` منفصلاً ومطابقاً للأصل على A4.
4. أضِف تفاعلات JS نظيفة + تحقق إدخال + حفظ مسودات.
5. اختبر: RTL، الطباعة (Preview)، الوصولية، التجاوب، وحدة console بلا أخطاء.

## فحوص القبول (Checklist) ✅
- [ ] HTML دلالي و`<h1>` واحد وتسلسل عناوين سليم.
- [ ] CSS بخصائص منطقية (يعمل RTL تلقائياً).
- [ ] الطباعة تطابق النموذج الأصلي على A4 بلا تقطيع.
- [ ] WCAG 2.2 AA (تباين/تركيز/كيبورد/ARIA).
- [ ] لا أخطاء console، أداء جيد، تجاوب كامل.
