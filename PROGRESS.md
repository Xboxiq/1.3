## ملفات المشروع الأساسية (مراجع سريعة)
- `data/services.json` — مخططات الخدمات (مصدر الحقيقة) + قوالب مشتركة `common.*` + بيانات الشركة.
- `assets/js/app.js` — محرّك العرض (Form Renderer) + المنطق + التحقق + الحفظ التلقائي.
- `assets/css/styles.css` — نظام التصميم (Design Tokens) + الشاشة + الوضع الليلي.
- `assets/css/print.css` — طباعة A4/RTL مطابقة للأصل.
- `index.html` — الواجهة الرئيسية (الأقسام الأربعة + البحث + المعاينة).
- `data/flowcharts/` — صور المخططات الانسيابية.
- `_src/` — ملفات المصدر المستخرجة + سكربتات استخراج docx (`docx_extract.py`, `docx_cells.py`). أداة عمل، ليست جزء المنتج.

## كيفية الاستخراج للخدمات اللاحقة (لتوفير الوقت)
```
cd "_src/نماذج طلبات  خدمات المشتركين/<القسم>/<مجلد الخدمة>"
python3 ../../../docx_cells.py "<اسم النموذج>.docx"     # بنية الجداول + التظليل
python3 ../../../docx_extract.py "<الوصف الإجرائي>.docx" # خطوات التنفيذ
```
